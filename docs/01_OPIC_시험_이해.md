# OPIC 시험 이해 문서

작성일: 2026-06-04

## 1. OPIC이란?

OPIC은 `Oral Proficiency Interview - computer`의 약자로, 컴퓨터로 진행하는 외국어 말하기 인터뷰 시험이다. 한국 공식 OPIC 사이트는 OPIC을 ACTFL이 개발, 평가, 인증하는 국제 공인 외국어 말하기 시험으로 설명하며, 실제 생활과 비즈니스 상황에서 외국어를 얼마나 사용할 수 있는지 평가한다고 안내한다.

핵심은 문법 시험이 아니라 `말로 의사소통이 되는가`를 보는 시험이다. 완벽한 영어보다, 질문을 이해하고 짧더라도 문장으로 대답하는 능력이 중요하다.

## 2. 시험 기본 정보

| 항목 | 내용 |
|---|---|
| 시험 방식 | 컴퓨터 기반 1:1 인터뷰 형식 |
| 시험 언어 | 영어, 중국어, 러시아어, 스페인어, 한국어, 일본어, 베트남어 등 |
| 문항 수 | 12~15문항 |
| 시험 시간 | 본시험 40분, OT 포함 약 60분 |
| 응시료 | 84,000원, VAT 포함 |
| 성적 발표 | 일반적으로 응시일 5일 후 |
| 평가 기준 | ACTFL Proficiency Guidelines 기반의 종합 평가 |

주의: 응시료, 성적 발표 일정, 센터 운영은 변경될 수 있으므로 시험 접수 전 공식 사이트에서 다시 확인한다.

## 3. 시험 진행 흐름

OPIC은 크게 오리엔테이션과 본시험으로 진행된다.

1. `Background Survey`: 관심사, 생활, 취미, 직장/학교 등 설문 선택
2. `Self Assessment`: 본인 말하기 수준 선택
3. `Pre-Test Setup`: 질문 듣기와 녹음 기능 점검
4. `Sample Question`: 화면 구성과 답변 방식 연습
5. `1st Session`: 개인 맞춤형 문항 약 7개
6. `난이도 재조정`: 쉬운/비슷한/어려운 질문 중 선택
7. `2nd Session`: 개인 맞춤형 문항 약 5~8개

공식 사이트 기준으로 질문은 2회 들을 수 있고, 문항별 답변 시간 제한은 없다. 다만 전체 본시험 시간은 40분이므로 초급자는 문항당 40초~70초 정도를 목표로 연습하는 것이 좋다.

## 4. Background Survey가 중요한 이유

OPIC은 사전 설문을 바탕으로 개인 맞춤형 질문을 낸다. 영어 초보자가 IL을 목표로 할 때는 실제 경험이 있고 말하기 쉬운 주제를 선택해야 한다.

추천 선택 원칙:

- 실제로 한국어로도 설명하기 쉬운 주제만 고른다.
- 영어 단어를 많이 모르는 분야는 피한다.
- 같은 문장 구조를 반복해서 쓸 수 있는 주제를 고른다.
- 집, 가족, 카페, 음악, 영화, 산책, 쇼핑, 집안일처럼 일상 단어로 말할 수 있는 주제가 좋다.

초보자 추천 주제:

| 영역 | 추천 |
|---|---|
| 거주 | 집, 방, 동네 |
| 여가 | 음악 듣기, 영화 보기, 카페 가기, 산책 |
| 운동 | 걷기, 조깅 |
| 휴가 | 국내 여행, 집에서 쉬기 |
| 일상 | 쇼핑, 음식 주문, 약속 |

## 5. 등급 체계

OPIC 등급은 낮은 순서부터 대략 다음과 같다.

`NL -> NM -> NH -> IL -> IM1 -> IM2 -> IM3 -> IH -> AL`

한국 공식 OPIC 사이트의 Intermediate Low 설명은 일상적인 소재에서는 문장으로 말할 수 있고, 선호하는 소재에서는 자신감을 가지고 대화에 참여할 수 있는 수준이다.

ACTFL Familiarization Guide에 따르면 Intermediate 수준은 예측 가능한 주제에서 짧은 대화에 참여하고, 간단한 질문에 묻고 답하며, 문장 단위로 의사소통할 수 있는 수준이다.

## 6. IL 목표자의 현실적인 기준

IL은 고급 영어가 아니다. 목표는 다음이다.

- 단어만 나열하지 않고 짧은 문장으로 말하기
- `I like`, `I go`, `I usually`, `It is`, `There is` 같은 기본 문장 사용
- 질문 주제에서 벗어나지 않기
- 3~6문장 정도로 답변 만들기
- 발음이 완벽하지 않아도 천천히 또박또박 말하기
- 모르는 질문이 나와도 외운 기본 구조로 버티기

IL 답변 예시:

```text
I live in an apartment.
My home is small but clean.
There is a bedroom, a living room, and a kitchen.
I like my room because it is quiet.
I usually watch TV at home.
```

이 정도의 짧고 쉬운 문장을 안정적으로 말하는 것이 우선이다.

## 7. 평가 요소

OPIC은 답변 하나하나를 점수화하기보다 전체 답변을 종합적으로 평가한다.

| 평가 요소 | 의미 | 초보자 전략 |
|---|---|---|
| Function | 질문에 맞는 기능 수행: 묘사, 설명, 질문, 요청 등 | 질문 유형별 고정 틀 암기 |
| Accuracy | 이해 가능한 발음, 문법, 어휘, 유창성 | 쉬운 문장으로 정확히 말하기 |
| Content/Context | 질문 의도와 맥락에 맞는 내용 | 모르는 말보다 아는 말로 주제 유지 |
| Text Type | 발화량과 답변 구조 | 3~6문장 템플릿 반복 |

## 8. 초보자가 조심할 점

- 어려운 단어를 억지로 쓰지 않는다.
- 긴 문장을 만들다가 멈추지 않는다.
- 질문을 완전히 못 알아들어도 침묵하지 않는다.
- 한국어식 발음이어도 괜찮으니 문장 끝까지 말한다.
- 외운 스크립트가 질문과 다르면 바로 쉬운 일반 문장으로 바꾼다.
