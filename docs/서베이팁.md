# OPIC 서베이팁

작성일: 2026-06-05

이 문서는 사용자가 제공한 2026년 OPIC 서베이 자료를 바탕으로, IL 목표 초보자가 시험장에서 어떤 항목을 고르면 준비 범위를 줄일 수 있는지 정리한 문서다. 실제 시험 화면의 항목명은 시기와 시험장 화면에 따라 조금 다르게 보일 수 있으므로, 최종 선택은 시험장에서 보이는 문구를 기준으로 한다.

## 1. 핵심 결론

IL 목표라면 서베이는 `내가 좋아하는 것`보다 `쉬운 영어로 말할 수 있는 것`을 고르는 것이 중요하다.

가장 중요한 기준:

1. 최소 12개 이상 선택한다.
2. 난이도는 IL 목표 기준 `3-3`을 우선 추천한다.
3. 직업, 학교, 수업처럼 설명이 어려운 주제는 줄인다.
4. 공원, 걷기, 음악, 영화, 쇼핑, 카페, 집 휴가처럼 쉬운 단어로 말할 수 있는 항목을 묶는다.
5. 선택 항목을 많이 늘리기보다, 한 답변 패턴으로 여러 항목을 처리할 수 있게 고른다.

## 2. 난이도 선택 요령

OPIC은 처음에 난이도를 선택하고, 시험 중간에 난이도 재조정 화면이 나온다.

난이도 흐름:

| 선택 | 의미 |
|---|---|
| 1~2 | 비교적 쉬운 질문, 약 12문항으로 알려져 있음 |
| 3~6 | 일반적으로 약 15문항으로 알려져 있음 |
| 중간 재조정 | 더 쉬운 질문, 비슷한 질문, 더 어려운 질문 중 선택 |

IL 목표 추천:

| 목표 | 추천 |
|---|---|
| IL | 처음 난이도 3 선택 후, 중간 조정에서 비슷한 질문 선택 |
| IM 이상 | 3-3, 4-4, 5-5를 목표와 실력에 맞게 선택 |

우리 프로젝트는 영어 초보 IL 취득이 목표이므로 `난이도 3-3`을 기본 전략으로 둔다.

## 3. 서베이 선택 원칙

서베이는 질문 소재를 정하는 과정이다. 예를 들어 공원 가기를 선택하면 공원 묘사, 공원 루틴, 공원 과거 경험 같은 질문이 나올 수 있다.

좋은 선택:

- 실제로 해본 적이 있다.
- 장소를 쉽게 설명할 수 있다.
- 보통 언제 하는지 말할 수 있다.
- 최근 또는 과거 경험을 짧게 말할 수 있다.
- 다른 항목과 같은 답변을 돌려쓸 수 있다.

피하고 싶은 선택:

- 한국어로도 설명하기 어렵다.
- 전문 단어가 많다.
- 규칙, 장비, 절차를 설명해야 한다.
- 나만의 경험이 별로 없다.
- 한 항목마다 완전히 다른 스크립트가 필요하다.

## 4. 전체 서베이 리스트

아래는 사용자가 제공한 두 자료를 분리해서 정리한 내용이다. 두 자료는 큰 구조는 같지만, 세부 항목명과 분류가 조금 다르다. 실제 시험에서는 시험장 화면의 항목명을 기준으로 선택한다.

| 대분류 | 1번 자료 | 2번 자료 |

| 취미/관심사 | 외국어 공부, 영화 리뷰, 수집하기 있음 | 외국어 공부, 영화 리뷰, 수집하기 없음 |

### 4-4. 우리 대시보드 반영 기준

대시보드는 전체 서베이 항목을 모두 외우는 도구가 아니다. 영어 초보 IL 목표에 맞게 실제로 공부할 항목만 약 20개 안팎으로 줄여서, 아래 5개 추천 조합과 돌발/위기탈출만 훈련한다.

| 대시보드 조합 | 포함 항목 |
|---|---|
| 공원/운동 | 공원 가기, 걷기, 조깅, 자전거 |
| 집/휴식 | 집에서 보내는 휴가, TV 시청, 게임하기 |
| 음악/영상 | 음악 감상하기, 영화보기, 콘서트 보기 |
| 카페/쇼핑 | 카페 가기, 쇼핑하기, 술집/바에 가기 |
| 여행 | 국내 여행, 해변 가기, 사진 촬영하기 |
| 돌발 유형 | 서베이와 무관하게 나올 수 있는 기술, 장소, 문제 상황, 롤플레이 |
| 위기탈출 | 질문을 못 들었거나 답변이 기억 안 날 때 쓰는 공통 대처 표현 |

아래는 두 자료를 합쳐 초보자 판단 기준으로 다시 정리한 상세 설명이다.

### Part 1. 직업

질문: 현재 귀하는 어느 분야에 종사하고 계십니까?

| 항목 | 초보자 판단 |
|---|---|
| 사업/회사 | 실제 직장 이야기를 준비할 수 있으면 가능 |
| 재택근무/재택 사업 | 루틴 설명이 가능하면 선택 가능 |
| 교사/교육자 | 설명할 단어가 늘어날 수 있어 주의 |
| 일 경험 없음 | 문제 범위를 줄이기 쉬워 추천 |

IL 목표 전략:

- 시험 전략만 보면 `일 경험 없음`이 가장 쉽다.
- 다만 실제 직장 답변을 이미 외우고 있다면 `사업/회사`도 가능하다.
- 사용자는 삼성전자 근무 표현을 이미 준비하고 있으므로, 직장 스크립트를 계속 쓸 계획이면 회사원 선택도 고려할 수 있다.

### Part 2. 학생 여부

질문: 현재 당신은 학생입니까?

| 항목 | 초보자 판단 |
|---|---|
| 예 | 학교, 수업, 숙제, 교수 관련 질문 가능 |
| 아니요 | 학교 관련 질문을 줄이기 쉬워 추천 |

IL 목표 전략:

- `아니요`를 추천한다.

### Part 2. 최근 수강 강의

질문: 최근 어떤 강의를 수강했습니까?

| 항목 | 초보자 판단 |
|---|---|
| 학위 과정 수업 | 과제, 수업, 전공 설명이 필요할 수 있음 |
| 전문 기술 향상을 위한 평생 학습 | 수업 내용 설명이 필요할 수 있음 |
| 어학 수업 | 영어 공부 답변과 연결 가능하지만 질문 범위가 늘어남 |
| 수강 후 5년 이상 지남 | 수업 관련 질문을 줄이기 쉬워 추천 |

IL 목표 전략:

- `수강 후 5년 이상 지남`을 추천한다.

### Part 3. 거주 형태

질문: 현재 귀하는 어디에 살고 계십니까?

| 항목 | 초보자 판단 |
|---|---|
| 개인주택이나 아파트에 홀로 거주 | 가족 질문을 줄이기 쉬워 추천 |
| 친구나 룸메이트와 함께 거주 | 집/동네 설명으로 연결 가능 |
| 가족과 함께 거주 | 가족 질문이 나올 수 있으나 실제 답변이 있으면 가능 |
| 학교 기숙사 | 학교 생활 질문으로 이어질 수 있음 |
| 군대 막사 | 특수 단어가 필요할 수 있어 비추천 |

IL 목표 전략:

- 문제 범위를 줄이려면 `개인주택이나 아파트에 홀로 거주`가 쉽다.
- 사용자는 가족 소개 문장을 이미 가지고 있으므로, 실제와 맞추고 싶다면 `가족과 함께 거주`도 준비 가능하다.
- 단, 가족 관련 비교/과거 경험/롤플레이가 부담되면 독신 거주 쪽이 더 안전하다.

### Part 4. 여가 활동

질문: 귀하는 여가 활동으로 주로 무엇을 하십니까? 두 개 이상 선택.

| 항목 | 추천도 | 이유 |
|---|---|---|
| 영화보기 | 추천 | 쉬운 감상 표현으로 답변 가능 |
| 클럽/나이트클럽 가기 | 주의 | 분위기, 사람, 경험 설명이 어려울 수 있음 |
| 박물관 가기 | 주의 | 작품, 전시 단어가 필요 |
| 공원 가기 | 추천 | 공원, 걷기, 휴식 표현으로 돌려쓰기 좋음 |
| 스포츠 관람 | 주의 | 팀, 경기, 규칙 설명이 필요할 수 있음 |
| 주거 개선 | 주의 | 수리, 인테리어 단어가 어려움 |
| 해변 가기 | 가능 | 여행/휴가와 묶으면 가능 |
| 요리 관련 프로그램 시청 | 주의 | 음식/요리 단어가 늘어남 |
| 게임하기 | 추천 | 집에서 하는 취미로 짧게 말하기 좋음 |
| 공연 보기 | 가능 | 영화/콘서트와 묶을 수 있음 |
| SNS에 글 올리기 | 주의 | 온라인 활동 설명 필요 |
| 캠핑 하기 | 비추천 | 장비, 날씨, 장소 단어가 어려움 |
| 구직 활동하기 | 비추천 | 직업/면접 관련 단어가 어려움 |
| 술집/바에 가기 | 가능 | 카페/친구 만남으로 단순화 가능 |

## 5. IL 목표 추천 조합

아래 조합은 질문 범위를 줄이고, 공통 답변을 많이 돌려쓰기 위한 조합이다.

### 안전 기본 선택

| 분류 | 추천 선택 |
|---|---|
| 직업 | 일 경험 없음 |
| 학생 | 아니요 |
| 최근 강의 | 수강 후 5년 이상 지남 |
| 거주 | 개인주택이나 아파트에 홀로 거주 |

이 네 가지는 직장, 학교, 수업, 가족 관련 질문 범위를 줄이는 데 도움이 된다.

### 12개 채우기 추천 조합

| 묶음 | 추천 항목 | 공통 답변 방향 |
|---|---|---|
| 공원/운동 | 공원 가기, 걷기, 조깅, 자전거 | `공원에 간다 → 걷는다 → 음악을 듣는다 → 편안하다` |
| 집/휴식 | 집에서 보내는 휴가, TV 시청, 게임하기 | `집에서 쉰다 → 게임/영화/음악을 한다 → 스트레스가 풀린다` |
| 음악/영상 | 음악 감상하기, 영화보기, 콘서트 보기 | `듣고 본다 → 재미있다 → 기분이 좋아진다` |
| 카페/쇼핑 | 카페 가기, 쇼핑하기, 술집/바에 가기 | `집 근처에 간다 → 친구/혼자 간다 → 음료를 마신다` |
| 여행 | 국내 여행, 해변 가기, 사진 촬영하기 | `부산에 갔다 → 바다를 봤다 → 사진을 찍었다` |

초보자용 실제 선택 예시:

1. 일 경험 없음
2. 학생 아니요
3. 수강 후 5년 이상 지남
4. 개인주택이나 아파트에 홀로 거주
5. 공원 가기
6. 영화보기
7. 쇼핑하기
8. 카페/커피전문점 가기
9. 게임하기
10. 음악 감상하기
11. 걷기
12. 집에서 보내는 휴가

추가로 여유가 있으면:

- 국내 여행
- 사진 촬영하기
- TV 시청하기
- 자전거

## 6. 나에게 맞춘 선택 방향

사용자의 현재 준비 표현을 기준으로 보면 아래 주제가 가장 잘 맞는다.

| 주제 | 이유 |
|---|---|
| 공원 가기 | 이미 `Whenever I have free time, I go to a park.` 표현을 사용 중 |
| 걷기 | 공원 답변과 완전히 같은 패턴 사용 가능 |
| 음악 감상 | 공원, 집, 출퇴근, 휴식 답변에 모두 붙일 수 있음 |
| 게임하기 | 이미 취미 표현으로 준비 중 |
| 집에서 보내는 휴가 | 게임, 음악, 영화 답변과 연결 가능 |
| 영화보기 | 쉬운 감상 표현으로 답변 가능 |
| 카페 가기 | 집 근처 장소 묘사와 루틴으로 말하기 쉬움 |
| 쇼핑하기 | 카페/주말/최근 활동과 묶기 좋음 |

사용자 맞춤 공통 표현:

```text
Whenever I have free time, I go to a park.
Walking in the park is one of my hobbies.
The park is quiet and clean.
I usually go there after work.
It makes me feel good and relaxed.
```

이 패턴은 공원, 걷기, 조깅, 자전거, 음악 감상, 휴식 질문에 거의 그대로 활용할 수 있다.

## 8. 서베이와 문제 패턴 연결

서베이 항목은 질문의 `소재`이고, 묘사/루틴/과거경험/비교/롤플레이는 답변의 `패턴`이다.

예: 공원 가기 선택 시

| 문제 패턴 | 질문 예시 | 답변 방향 |
|---|---|---|
| 묘사 | Describe a park near your home. | 공원이 조용하고 깨끗하다 |
| 루틴 | How often do you go to the park? | 퇴근 후 자주 간다 |
| 과거 경험 | Tell me about the last time you went there. | 지난 주말에 갔다 |
| 비교 | Compare this park with another park. | 이 공원이 더 조용하다 |
| 의견 | Why do you like going to the park? | 편안해서 좋다 |
| 돌발 | Tell me about a problem at a park. | 사람이 많아서 불편했다 |

따라서 대시보드에서는 서베이 항목을 먼저 고르고, 그 아래에서 묘사/루틴/과거경험 같은 질문 패턴을 랜덤으로 연습한다.

## 9. 시험장에서 기억할 말

1. 어려운 항목을 고르면 어려운 질문을 받을 수 있다.
2. 12개는 채우되, 서로 비슷한 항목으로 채운다.
3. 공원, 걷기, 음악, 집 휴가, 영화, 카페, 쇼핑, 게임은 초보자에게 유리하다.
4. 직장/학교/수업은 준비가 없으면 줄이는 쪽이 쉽다.
5. 모든 답변은 `장소 + 보통 하는 일 + 이유 1개`로 끝낸다.

가장 중요한 한 줄:

```text
쉬운 주제를 고르고, 같은 문장을 계속 돌려쓴다.
```
