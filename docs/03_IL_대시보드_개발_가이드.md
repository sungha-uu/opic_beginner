# OPIC IL 학습 대시보드 개발 가이드

작성일: 2026-06-04

## 1. 제품 목표

이 프로젝트의 최종 목적은 영어 초보자가 OPIC IL을 취득할 수 있도록 매일 해야 할 공부를 보여주고, 녹음/복습/시험 결과를 추적하는 학습 대시보드를 만드는 것이다.

사용자는 영어를 거의 못하는 상태이므로 앱은 복잡한 학습 관리 도구가 아니라 `오늘 무엇을 말하면 되는지`를 바로 보여주는 도구여야 한다.

2026-06-04 업데이트: 현재 제품 방향은 일반 학습 관리가 아니라 `OPIC 유형별 맞춤 스크립트 암기 훈련장`이다. 오늘의 준비도, 오늘 할 일, 오늘의 고정 표현, 시험 일정, 다음 보완 메모, 오늘 외울 문장은 제거하고, 맞춤 스크립트와 60초 랜덤 연습에 집중한다.

질문 데이터는 공식 기출 원문 보장이 아니라, 공개 샘플과 응시 후기/학습 자료에서 반복적으로 보고되는 실전 패턴 기반 예상 질문으로 구성한다. 실제 시험처럼 느끼도록 질문 수는 넓게 가져가되, 답변 스크립트는 IL 목표에 맞춰 공통 표현을 반복한다.

## 2. 핵심 사용자

사용자 프로필:

- 한국어 사용자
- 영어 기초 매우 부족
- OPIC IL 취득이 반드시 필요함
- 한 달에 한 번 정도 시험 응시 예정
- 리스닝과 스피킹에 두려움이 있음

UX 원칙:

- 첫 화면에서 유형별 맞춤 스크립트를 바로 보여준다.
- 영어 문장은 짧고 크게 보여준다.
- 영어 답변에는 한글 발음을 붙이되, 발음 표시는 작고 얇게 보여준다.
- 점수보다 유형별 질문 반복과 스크립트 암기를 강조한다.

## 3. MVP 범위

1차 버전에서 반드시 필요한 기능:

- OPIC 유형별 맞춤 스크립트 목록
- 유형별 다양한 실전형 예상 질문
- 유형 탭 선택 시 해당 유형 질문 5개를 랜덤 표시
- 위기탈출 스크립트
- 60초 랜덤 질문 연습
- 60초 동안 질문만 표시
- 60초 후 유형, 질문 해석, 좋은 답변 예시 표시
- 연습 중 종료 버튼으로 즉시 답변 표시
- 문장별 영어와 한글 발음 표시

나중에 추가할 기능:

- 음성 인식 STT
- 발음 피드백
- AI 답변 첨삭
- 녹음 저장
- 유형별 암기 완료 표시
- 질문별 반복 횟수 기록

## 4. 화면 구성

### Scripts

첫 화면. OPIC 유형별 답변 스크립트를 관리한다.

구성:

- 유형 필터. `전체` 탭은 두지 않는다.
- 선택한 유형의 질문 5개를 랜덤 표시
- 다른 유형을 눌렀다가 돌아오면 질문 5개를 다시 랜덤 표시
- 개인 맞춤 답변
- 문장별 한글 발음
- 위기탈출 유형 포함

### Practice

두 번째 화면. 랜덤 60초 연습을 제공한다.

구성:

- 60초 연습 버튼
- 버튼 클릭 시 랜덤 유형/랜덤 질문 표시
- 연습 중에는 질문만 표시
- 연습 중 버튼은 빨간색 `종료`로 변경
- `종료` 클릭 시 즉시 유형, 질문 해석, 좋은 답변 예시 표시
- 60초가 지나도 동일하게 답변 표시
- 답변 표시 후 버튼은 파란색 `60초 연습`으로 복귀

## 6. 학습 점수 계산

대시보드의 점수는 실제 OPIC 점수가 아니라 학습 진행도를 보여주는 내부 지표다.

Daily Readiness Score 예시:

| 항목 | 배점 |
|---|---|
| 오늘 체크리스트 완료 | 30 |
| 녹음 2개 이상 | 25 |
| 스크립트 1개 암기 | 20 |
| 단어 10개 복습 | 15 |
| 자기 평가 통과 | 10 |

합계 100점.

표시 방식:

- 0~39: 오늘은 최소 녹음 1개라도 하기
- 40~69: 기본 루틴 완료
- 70~89: 시험 준비 양호
- 90~100: 실전 루틴 우수

## 7. 개인 맞춤 스크립트 생성 규칙

앞으로 스크립트 데이터를 만들 때는 사용자가 이미 익숙한 표현을 우선 반영한다. 목표는 새로운 고급 표현을 많이 넣는 것이 아니라, 익숙한 쉬운 문장을 OPIC 채점자가 이해하기 쉬운 형태로 고쳐 반복시키는 것이다.

우선 사용할 핵심 문장:

```text
Whenever I have free time, I go to a park.
Walking in the park is one of my hobbies.
Hello, my name is OOO.
I am 40 years old.
I work for Samsung Electronics.
I have worked there for 10 years.
There are four people in my family.
I live in Daegu, Korea.
Playing games is one of my hobbies.
These days, I study English after work.
```

스크립트 생성 원칙:

- 자기소개, 취미, 자유시간, 공원, 게임, 직장 주제에서는 위 문장을 우선 재사용한다.
- `one of my hobbies`, `It helps me relax`, `I usually ... after work`를 반복 패턴으로 쓴다.
- `because` 뒤에는 항상 완전한 문장을 붙인다.
- 초보자용 예문은 5~7문장을 넘기지 않는다.
- 어려운 접속사보다 `and`, `but`, `because`, `so`를 우선 사용한다.

예시 데이터:

```json
{
  "id": "script_personal_free_time_001",
  "topicId": "topic_free_time",
  "question": "What do you usually do in your free time?",
  "answer": [
    "Whenever I have free time, I go to a park.",
    "Walking in the park is one of my hobbies.",
    "The park is quiet and clean.",
    "I usually walk there after work.",
    "It helps me relax."
  ],
  "translationKo": [
    "저는 시간이 날 때마다 공원에 갑니다.",
    "공원에서 걷는 것은 제 취미 중 하나입니다.",
    "그 공원은 조용하고 깨끗합니다.",
    "저는 보통 퇴근 후에 거기서 걷습니다.",
    "그것은 제가 쉬는 데 도움이 됩니다."
  ],
  "targetLevel": "IL",
```

## 12. 실전 연습 플로우

1. 사용자는 먼저 `Scripts` 화면에서 유형별 맞춤 스크립트를 암기한다.
2. 유형 탭을 누르면 해당 유형의 질문 5개가 랜덤으로 표시된다.
3. 사용자는 질문 패턴을 보고 같은 답변 스크립트를 반복해서 읽는다.
4. `Practice` 화면에서 `60초 연습` 버튼을 누른다.
5. 앱은 랜덤 유형과 랜덤 질문을 선택하고, 질문만 보여준다.
6. 버튼은 빨간색 `종료`로 바뀐다.
7. 사용자는 60초 동안 질문만 보고 말한다.
8. 60초가 지나거나 `종료` 버튼을 누르면 유형, 질문 해석, 좋은 답변 예시가 표시된다.
9. 버튼은 다시 파란색 `60초 연습`으로 돌아간다.

## 13. 디자인 방향

OPIC 초보자용 학습 앱은 불안감을 줄이는 UI가 중요하다.

방향:

- 복잡한 차트보다 유형별 스크립트와 질문을 선명하게 보여준다.
- 색상은 차분하되 성공 상태는 명확하게 표시한다.
- 버튼 문구는 짧게 쓴다.
- 영어 답변과 한글 발음을 함께 보여준다.
- 한글 발음은 주황색이지만 작고 얇게 표시해 시각적 부담을 줄인다.
- 카드 안에 카드를 넣지 않는다.
- 모바일에서도 60초 연습 버튼이 쉽게 눌려야 한다.

## 14. 개발 단계

### Phase 1: 문서와 데이터

- 현재 문서 3종 완성
- OPIC 유형 정의
- 유형별 질문 풀 생성
- 유형별 맞춤 스크립트 생성
- 위기탈출 스크립트 생성

### Phase 2: 정적 대시보드

- 맞춤 스크립트 화면
- 유형별 랜덤 질문 5개 표시
- 60초 랜덤 연습
- 즉시 종료 버튼

### Phase 3: 녹음 기반 연습

- 녹음 기능
- 재생 기능
- 자기 평가
- 학습 기록 저장

### Phase 4: 시험 추적

- 유형별 반복 횟수 기록
- 자주 틀리는 질문 기록
- 위기탈출 사용 횟수 기록
- 다음 암기 우선순위 추천

### Phase 5: AI 보조 기능

- 답변 문장 쉬운 영어로 변환
- 발화량 체크
- 답변이 질문과 맞는지 확인
- 초보자용 피드백 생성

## 15. 성공 기준

MVP 성공 기준:

- 사용자가 앱을 열고 바로 맞춤 스크립트를 볼 수 있다.
- 사용자가 유형 탭을 누를 때마다 질문 5개를 새롭게 확인할 수 있다.
- 사용자가 1분 안에 60초 랜덤 연습을 시작할 수 있다.
- 사용자가 연습 중 종료 버튼으로 즉시 답변 예시를 확인할 수 있다.

학습 성공 기준:

- 자기소개를 30초 이상 말할 수 있다.
- 각 유형별 맞춤 스크립트를 5문장 이상 말할 수 있다.
- 롤플레이에서 질문 또는 요청 문장을 3개 이상 말할 수 있다.
- 아무 생각이 나지 않을 때 위기탈출 스크립트를 말할 수 있다.
- 15문항 모의고사에서 침묵 없이 끝까지 답변할 수 있다.
