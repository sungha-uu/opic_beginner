# OPIC Master Agent Guide

이 프로젝트는 영어 초보 학습자의 OPIC IL 취득을 지원하는 정적 웹 대시보드다. 변경 전 `README.md`, `docs/`, 기존 UI 구조를 확인한다. 학습 자료는 초보자가 발음하고 암기할 수 있는 난이도를 유지하며, 시험 결과를 보장하는 표현은 사용하지 않는다. 변경 후 주요 화면을 브라우저에서 확인한다.

## `#마스터` 호출

사용자 요청이 `#마스터`로 시작하면 `D:\WorkSpace\Codex\_MasterAgent\MASTER_PROTOCOL.md` 형식으로 마스터 스레드 `019eb6c7-f75d-7971-89ee-586e4b5fc1e1`에 전달한다. 직접 전달할 수 없으면 전달용 요약을 사용자에게 제공한다.
