# OPIC Master Agent Guide

이 프로젝트는 영어 초보 학습자의 OPIC IL 취득을 지원하는 정적 웹 대시보드다. 변경 전 `README.md`, `docs/`, 기존 UI 구조를 확인한다. 학습 자료는 초보자가 발음하고 암기할 수 있는 난이도를 유지하며, 시험 결과를 보장하는 표현은 사용하지 않는다. 변경 후 주요 화면을 브라우저에서 확인한다.

## 콘텐츠 운영

- 웹페이지는 학습과 열람 전용이며 콘텐츠 CRUD UI를 추가하지 않는다.
- 질문, 답변 패턴, 필러는 `app.js`의 `sourceQuestions`, `sourceAnswerPatterns`, `sourceFillers`를 기준 원천으로 사용한다.
- 실제 시험 질문은 사용자가 명시적으로 전달한 경우에만 `actual`로 등록한다. 에이전트가 만든 질문은 `expected`로 등록한다.
- 사용자의 기존 답변 패턴을 임의로 만들지 않는다. 사용자가 한국어 소재를 전달하면 원문을 보존하고 쉬운 영어로 정리해 추가한다.
- `personalNotes`는 독립 메모이며 추천 답변 계산에서 절대 읽지 않는다.
- 답변 패턴, 필러, 메모는 기본 접힘 상태를 유지한다. 질문은 선택한 유형 안에서 기본 펼침 상태를 유지한다.
