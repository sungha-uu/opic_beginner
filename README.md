# OPIC Personal Trainer

OPIC IL을 준비하는 영어 초보자가 시험에서 들은 질문과 자신의 영어 소재를 직접 쌓는 정적 웹 대시보드입니다. 시험 결과를 보장하지 않으며, 짧고 쉬운 문장을 반복해서 말하는 연습을 돕습니다.

## 대시보드 실행

브라우저에서 [index.html](index.html)을 열면 사용할 수 있습니다. 별도 서버나 빌드 과정은 필요하지 않습니다.

## P07 제품 구조

- `질문 패턴`: 사용자가 실제 시험에서 들은 질문은 `기출 질문`, 기출을 바탕으로 만든 연습 문장은 `예상 질문`으로 저장합니다.
- `답변 패턴`: 한국어 개인 이야기와 말하기 쉬운 영어 정리문을 경험, 취향, 루틴, 장소, 사람 등의 재사용 가능한 소재로 저장합니다.
- `추천 답변`: 예상 질문에 개인 답변 패턴을 붙이고 질문 유형에 맞는 쉬운 문장을 추가합니다. 추천 근거를 카드에 표시합니다.
- `60초 연습`: 저장된 기출/예상 질문 중 하나를 무작위로 연습합니다.
- `필러`: 연결, 생각할 시간, 예시, 정정, 마무리 표현을 직접 추가·수정·삭제합니다.
- `메모장`: 기존 개인 메모를 유지하지만 답변 패턴과 추천 답변의 입력 데이터로 사용하지 않습니다.

질문, 답변 패턴, 필러는 브라우저 `localStorage`에 저장됩니다. 기존 메모 삭제 정보는 호환을 위해 `opicDeletedMemoIds` 키를 계속 사용합니다.

## `#마스터 P07` 운영

사용자는 마스터에게 자연어로 기출 질문이나 한국어 답변 소재를 전달할 수 있습니다.

```text
#마스터 P07
기출 질문: 시험에서 최근 공원에 갔던 경험을 물어봤어요.
영어 기억: Tell me about the last time you went to a park.
질문 유형: 경험
```

```text
#마스터 P07
답변 소재: 저는 퇴근 후 집 근처 공원을 걷습니다. 공원이 조용해서 좋아합니다.
소재 종류: 일상 루틴
```

에이전트는 기출 질문을 `source: "actual"`로, 유사 연습 질문을 `source: "expected"`와 `originQuestionId`로 구분합니다. 답변 소재는 한국어 원문을 보존하고 초보자가 발음·암기할 수 있는 짧은 영어 문장을 줄 단위로 정리합니다.
