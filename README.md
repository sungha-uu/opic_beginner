## 대시보드 실행

브라우저에서 [index.html](index.html)을 열면 OPIC IL 스크립트 트레이너를 사용할 수 있습니다. 화면은 맞춤 스크립트, 60초 연습 순서로 구성되어 있습니다.

## 목표

유형별 다양한 OPIC 질문을 랜덤으로 연습하고, 개인 맞춤 스크립트와 위기탈출 스크립트를 반복 암기합니다.
