Tip

IL 준비 원칙

  1. IL 목표는 난이도 3을 우선 선택한다.
  2. 기본 문장으로 답변하고, 가능하면 `because`로 이유 1개를 붙인다.
  3. 문법보다 끊기지 않고 끝까지 말하는 흐름을 더 중요하게 연습한다.
  4. 답변은 `기본 문장 + 이유 1개` 구조로 외운다.
  5. 롤플레이는 상황 설명, 간단한 요청, 변경 질문을 30~40초로 연습한다.
  6. 서베이는 실제 경험이 있고 같은 표현으로 돌려 말할 수 있는 항목을 고른다.
  7. 추천 묶음은 공원/걷기/음악, 카페/쇼핑, 게임/음악, 국내여행/교통이다.

서베이와 문제 패턴 구분

서베이 주제공원, 카페, 게임, 여행, 직장처럼 질문 소재가 되는 항목
문제 패턴묘사, 루틴, 과거경험, 비교, 롤플레이처럼 답변 방식
추천 선택공원/걷기/음악처럼 한 답변으로 묶이는 항목
피할 선택악기, 캠핑, 해외여행처럼 단어가 어려운 항목
콤보 대비한 주제에서 묘사 → 루틴 → 과거경험으로 확장
위기탈출모르는 질문은 아는 경험으로 연결
